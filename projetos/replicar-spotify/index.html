Alguma dúvida ?

  • Como faço para criar uma playlist? ⇓

    As playlist são uma ótima maneira de criar coleções de músicas,seja para só você ouvir ou para compartilhar.

    Para criar uma:

    1.Clique na "Sua Biblioteca".
    2. Clique em CRIAR
    3.Dê um nome para a playlist.
    4. Comece a incluir algumas músicas (enquanto recomendamos outras).

  • Como faço para ativar o modo de Economia de dados? ⇓

    1. Clique em Início
    2. Clique em Configurações.
    3. Clique em Economia de dados.
    4. Ative a "Economia de dados."

  • Só é possivel ouvir música em ordem aleátoria? ⇓

    As playlist com o ícone de ordem aleátoria tocam dessa forma.

    Nas playlist que não têm esse ícone,você pode clicar em qualquer música para ouvir.

  • Onde posso encontra podcasts? ⇓

    Clique em Buscar.Em Navegar todas as seções, clique em podcasts.
